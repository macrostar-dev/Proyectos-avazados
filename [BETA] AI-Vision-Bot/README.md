Instrucciones de Configuración (Importante)
Para que el bot funcione correctamente y sea preciso, es necesario preparar el entorno de entrenamiento:
Descomprimir imágenes: Se debe extraer el contenido del archivo img.rar en la carpeta correspondiente del proyecto.
Mejorar la precisión: El modelo actual es una base; para obtener resultados más exactos, se recomienda agregar más imágenes a las carpetas de entrenamiento antes de exportar el modelo (.h5).
Entrenamiento: Cuantas más referencias visuales tenga el modelo sobre un objeto o categoría, menor será el margen de error en la predicción de Discord.

English
Setup Instructions (Important)
For the bot to work correctly and accurately, the training environment must be prepared:
Extract Images: You must extract the contents of the img.rar file into the corresponding project folder.
Improve Accuracy: The current model is a baseline; to achieve more accurate results, it is recommended to add more images to the training folders before exporting the model (.h5).
Training: The more visual references the model has for an object or category, the lower the error margin will be in the Discord prediction.
