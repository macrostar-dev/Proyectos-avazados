En español:

"Una plataforma web interactiva creada con Flask y SQLAlchemy. Es un proyecto de aprendizaje diseñado para gestionar perfiles profesionales por categorías (como Programación o Repostería). Aunque se basa en una estructura de estudio, personalicé la lógica de la base de datos para permitir el registro de redes sociales y visualización dinámica. Estado: Fase Beta (Prueba técnica)."

In English:

"An interactive web platform built with Flask and SQLAlchemy. This learning project is designed to manage professional profiles by categories (such as Programming or Baking). While based on a study structure, I customized the database logic to allow social media registration and dynamic display. Status: Beta Phase (Technical test)."
